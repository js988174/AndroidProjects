    ## Log 

        - 로그란 ? 
           + 로그를 사용하여 해당 코드의 흐름을 알기 위해서

            - Log.d("로그 검색용", "출력문 = 로그 내용")
            - v : verbose -> 상세한 로그 내용 출력
            - d : debug -> 개발에 필요한 내용 출력
            - i : information -> 일반적인 메시지 전달
            - w : warning -> 경고성 메시지 전달 
            - e : error -> 에러 메시지를 출력
